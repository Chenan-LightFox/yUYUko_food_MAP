**管理员后台权限相关**

管理员权限分为以下几点：

    1. 管理所有用户

    2. 管理一般用户（非管理员）

    3. 管理地图标记点

    4. 管理邀请码

    5. 管理评论

管理员分为以下几个级别：

    - 幽幽子（YUYUKO）Y：拥有以上所有权限

    - 妖梦（YOUMU）M：拥有以上权限2-5

    - 小町（KOMACHI）K：拥有以上权限2和5
